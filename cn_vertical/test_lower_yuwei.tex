\documentclass{guji}

% Test the lower-yuwei parameter
% 测试下鱼尾参数

\begin{document}

% Default behavior: lower yuwei is enabled (默认：显示下鱼尾)
\gujiSetup{
  book-name={四庫全書},
  chapter-title={史記\\卷一},
}

\begin{guji-content}
这是第一页的测试文本，使用默认设置，下鱼尾应该显示。
版心中应该有上下两个鱼尾装饰。
\end{guji-content}

% Disable lower yuwei (禁用下鱼尾)
\gujiSetup{
  book-name={四庫全書},
  chapter-title={史記\\卷二},
  lower-yuwei=false,
}

\begin{guji-content}
这是第二页的测试文本，禁用了下鱼尾。
版心中应该只有上鱼尾，没有下鱼尾。
\end{guji-content}

\end{document}
