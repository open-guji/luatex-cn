% example.tex
% Example document demonstrating luatex-cn package usage

\documentclass[12pt]{ctexart}
\usepackage[vertical,simplified]{luatex-cn}

% Set fonts
\setCJKmainfont{Noto Serif CJK SC}[
  UprightFont = *-Regular,
  BoldFont = *-Bold,
]

\title{中文竖排示例}
\author{luatex-cn 包示例}
\date{\today}

\begin{document}

\maketitle

\section{横排文本示例}

这是一个横排的中文文本示例。本包支持中文排版和竖排功能。

\section{竖排文本示例}

\begin{tate}
这是竖排的中文文本示例。
古籍排版通常使用竖排格式。
本包提供了对竖排的良好支持。
\end{tate}

\section{混合排版}

横排和竖排可以混合使用，满足不同的排版需求。

\begin{tate}
竖排文本可以包含标点符号，如：。，；：？！等。
\end{tate}

\end{document}
