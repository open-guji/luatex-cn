% 混合测试：专名号和书名号相邻
\专名号{司馬遷}撰\书名号{史記}混合

% 波浪线振幅测试（标准样式 - 默认）

\书名号[amplitude=medium]{中振幅}中

\书名号[amplitude=large]{大振幅}大

\书名号[style=cursive, amplitude=large]{大草书}大草
