天地玄黄，宇宙洪荒。日月盈昃，辰宿列张。寒來暑往，秋收冬藏。閏餘成歲，律呂調陽。

子曰：「學而時習之，不亦說乎？」有朋自遠方來，不亦樂乎？人不知而不慍，不亦君子乎？

「一二三」「四五六」七八九十。
