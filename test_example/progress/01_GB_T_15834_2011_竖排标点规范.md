# GB/T 15834―2011《标点符号用法》竖排标点规范

> **标准全称**: 中华人民共和国国家标准 GB/T 15834―2011《标点符号用法》
> **发布日期**: 2011-12-30 | **实施日期**: 2012-06-01
> **替代标准**: GB/T 15834-1995
> **适用范围**: 现代汉语书面语（包括汉语和外语混合排版时的汉语部分）

## 一、标准版本演变

| 版本 | 年份 | 主要变化 |
|------|------|----------|
| GB/T 15834-1995 | 1995 | 首次系统规定标点符号用法 |
| GB/T 15834-2011 | 2011 | 增加附录A（规范性）和附录B（资料性）；省略号统一为六连点"……"；取消二字线连接号；明确书名号使用范围 |

## 二、标点符号分类与基本形式

### 2.1 点号（8种）

| 名称 | 符号 | 占位 | 说明 |
|------|------|------|------|
| 句号 | 。 | 1字位 | 句末点号，标示陈述句末尾停顿 |
| 问号 | ？ | 1字位 | 句末点号，标示疑问句末尾语气 |
| 叹号 | ！ | 1字位 | 句末点号，标示感叹句末尾语气 |
| 逗号 | ， | 1字位 | 句内点号，标示句子内部一般停顿 |
| 顿号 | 、 | 1字位 | 句内点号，标示并列词语间停顿 |
| 分号 | ； | 1字位 | 句内点号，标示复句内部分句间停顿 |
| 冒号 | ： | 1字位 | 句内点号，标示提示性话语后停顿 |

### 2.2 标号（8种）

| 名称 | 符号 | 占位 | 说明 |
|------|------|------|------|
| 引号 | ""（双）''（单） | 各1字位 | 标示引用或特殊含义 |
| 括号 | （）〔〕［ ］【】 | 各1字位 | 标示注释性话语 |
| 破折号 | —— | **2字位** | 标示注释、补充说明或语音变化 |
| 省略号 | …… | **2字位** | 标示省略或断续 |
| 着重号 | · | 标注于文字下方 | 标示需要强调的文字 |
| 连接号 | -（半字位）—（1字位）~（1字位） | 见左 | 标示相关联成分间连接 |
| 间隔号 | · | 半字位 | 标示某些相关联成分间分界 |
| 书名号 | 《》（双）〈〉（单） | 各1字位 | 标示作品名称 |
| 专名号 | ＿ | 标注于文字下方 | 标示古籍中专有名词 |
| 分隔号 | / | 半字位 | 标示诗行、日期等分隔 |

## 三、第5章：标点符号位置和书写形式

### 3.1 横排文稿（§5.1）

| 条款 | 标点类型 | 位置规定 |
|------|----------|----------|
| 5.1.1 | 句号、逗号、顿号、分号、冒号 | 占1字位，**居左下**，不出现在一行之首 |
| 5.1.2 | 问号、叹号 | 占1字位，**居左**，不出现在一行之首 |
| 5.1.3 | 引号、括号、书名号 | 前后各占1字位；前一半不出现在行末，后一半不出现在行首 |
| 5.1.4 | 破折号 | 占2字位，**上下居中**，不能中间断开 |
| 5.1.5 | 省略号 | 占2字位；四个连用时占4字位，不能中间断开 |

### 3.2 竖排文稿（§5.2）⭐ 核心内容

| 条款 | 标点类型 | 位置规定 |
|------|----------|----------|
| **5.2.1** | 句号、问号、叹号、逗号、顿号、分号、冒号 | 均置于相应文字之下**偏右** |
| **5.2.2** | 破折号、省略号、连接号、间隔号、分隔号 | 置于相应文字之下**居中**，**上下方向排列** |
| **5.2.3** | 引号 → 改用 ﹃﹄（双）和 ﹁﹂（单）；括号 → 改用 ︵︶ | 标在相应项目的**上下** |
| **5.2.4** | 书名号 → 改用浪线式 ﹏ | 标在相应文字的**左侧** |
| **5.2.5** | 着重号 → 标在相应文字的**右侧**；专名号 → 标在相应文字的**左侧** |
| **5.2.6** | 关于某些标点不能居行首或行末的要求，同样适用于竖排文稿 |

## 四、竖排引号详细对照表

| 横排形式 | 竖排形式 | Unicode码位 | 名称 |
|----------|----------|-------------|------|
| "（左双引号） | ﹃ | U+FE43 | PRESENTATION FORM FOR VERTICAL LEFT WHITE CORNER BRACKET |
| "（右双引号） | ﹄ | U+FE44 | PRESENTATION FORM FOR VERTICAL RIGHT WHITE CORNER BRACKET |
| '（左单引号） | ﹁ | U+FE41 | PRESENTATION FORM FOR VERTICAL LEFT CORNER BRACKET |
| '（右单引号） | ﹂ | U+FE42 | PRESENTATION FORM FOR VERTICAL RIGHT CORNER BRACKET |
| （（左括号） | ︵ | U+FE35 | PRESENTATION FORM FOR VERTICAL LEFT PARENTHESIS |
| ）（右括号） | ︶ | U+FE36 | PRESENTATION FORM FOR VERTICAL RIGHT PARENTHESIS |

> **注意**: GB/T 15834-2011 中竖排双引号（﹃﹄）是否为实心角括号存在争议。知乎问答中有讨论：标准原文中的﹃﹄是空心的白角引号。

## 五、附录A（规范性附录）关键补充

| 条目 | 内容摘要 |
|------|----------|
| A.1 | 图表短语式说明文字末尾不用句号 |
| A.2 | 问号使用以语气为据，非以疑问词为据 |
| A.3 | 叹号：有关用法说明 |
| A.5 | 顿号：表顺序关系的并列项间用顿号不用逗号 |
| A.7 | 分号：分项列举含句号时各项末尾不能用分号 |
| A.10 | 省略号不能用多于两个连用；省略号和"等""等等"不能同时使用 |
| A.13 | 书名号：课程、课题、奖品、商标、证照、组织机构等名称**不应**用书名号 |

## 六、附录B（资料性附录）标点区分

| 条目 | 内容 |
|------|------|
| B.1.1 | **逗号 vs 顿号**: 逗号停顿长、顿号停顿短；并列词语间通常用顿号 |
| B.1.2 | 停顿长短顺序：句号＝问号＝叹号 ＞ 冒号 ＞ 分号 ＞ 逗号 ＞ 顿号 |
| B.1.7 | **破折号 vs 括号**: 破折号标示的注释是正文的一部分；括号标示的注释不是正文 |
| B.2 | 两个标点连用规则（引号内外点号用法） |
| B.3 | 序次语后的标点规则 |
| B.4 | 文章标题的标点规范 |

## 七、与竖排排版相关的其他要点

### 7.1 行首行末禁则（同时适用于竖排）

**禁止出现在行首**：
- 句号、问号、叹号、逗号、顿号、分号、冒号
- 引号后半部分（﹂﹄）、括号后半部分（︶）
- 书名号后半部分

**禁止出现在行末**：
- 引号前半部分（﹁﹃）、括号前半部分（︵）
- 书名号前半部分

### 7.2 不可断开的标点

- 破折号（——）：占两字位，中间不得断开
- 省略号（……）：占两字位，中间不得断开
