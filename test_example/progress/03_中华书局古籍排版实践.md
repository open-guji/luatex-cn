# 中华书局古籍排版体例与行业实践

> 本文整理中华书局及古籍出版行业在竖排标点符号方面的实际操作规范，
> 包括行业标准、古籍校点释例、以及排版工艺中的具体实践。

## 一、行业标准概览

### 1.1 相关标准文件

| 标准编号 | 名称 | 年份 | 说明 |
|----------|------|------|------|
| GB/T 15834-2011 | 标点符号用法 | 2011 | 国家标准，古籍整理的标点依据 |
| CY/T 124-2015 | 学术出版规范·古籍整理 | 2015 | 行业标准，规范古籍整理出版 |
| WH/T 102-2023 | 汉文古籍版式描述规范 | 2023 | 文化行业标准，描述古籍版式 |

### 1.2 全球汉籍合璧工程工作规程

该规程对古籍整理有详细规定：
- **标点原则**: 尽量多用逗号、句号，少用叹号、问号、分号
- **禁用标点**: 禁用省略号、破折号、专名号
- **引号规定**: 第一层用双引号（""），第二层用单引号（''）
- **书名号**: 五经、四书等统称不加书名号；简称需加号
- **排版要求**: 全部书稿用繁体字横排（此为该项目特例，非通则）

## 二、古籍校点标点体例

### 2.1 古籍校点释例中的标点规定

来源：中华书局古籍校点通则及相关释例

#### 句号（。）
- 用于陈述句末尾、语气舒缓的祈使句末尾
- **韵文押韵处**使用句号
- 文字脱漏导致语意不接时，宜用句号隔开

#### 问号（？）
- 用于疑问句末尾，反问句也用问号
- 有疑问词但不属疑问句的不用问号
- 不求回答的语句不用问号

#### 叹号（！）
- 用于感叹句末尾和语气强烈的祈使句
- **标点古籍中叹号不宜多用**，凡可用叹号处亦可用句号

#### 逗号（，）
- 表示句子内部一般停顿
- 判断句主语谓语间需用逗号点开
- 并列成分层次不同时，逗号和顿号相间使用

#### 顿号（、）
- 表示并列词语间的停顿
- 易引起误会的并列名词需用顿号
- 层次繁复的并列名词宜从简处置

#### 分号（；）
- 隔开复句内的并列分句
- **散文中应少用**
- **骈文中仅限对仗句式**

#### 冒号（：）
- 提示性话语后用冒号提起下文
- 表示说话的句子冒号下必有引号
- 总括性话语前可用冒号总结上文

#### 引号
- 直排第一层用单引号「」，第二层用双引号『』
- 引文末尾附加语气词时，下引号放在附加词之上

#### 括号
- **圆括号（）**: 表示删去
- **方括号〔〕**: 表示改正或增补
- 注码用圆括号或圈码，校记用方括号

#### 书名号（《》）
- 标明书名、篇名、乐舞名等
- 书名篇名间用间隔号（·）分界
- 同一书中不同篇名连用时处理方式有特殊规定

#### 专名号（＿）
- 标明人名、地名、朝代名、民族名、国名等
- **字号、封谥、尊称**等意在专指者一律标专名线

## 三、竖排标点在古籍中的具体位置规范

### 3.1 中华书局实际体例

| 符号 | 竖排位置 | 要求 | 备注 |
|------|----------|------|------|
| 句号 。 | 字格右上角 | **严禁居中或置于左侧** | 以免与注疏混淆 |
| 逗号 ， | 字格右上角 | 偏右 | |
| 顿号 、 | 字格右上角 | 偏右 | |
| 分号 ； | 字格右上角 | 偏右 | 散文少用 |
| 冒号 ： | 字格右上角 | 偏右 | |
| 问号 ？ | 居中 | | |
| 叹号 ！ | 居中 | 古籍中少用 | |
| 书名号 ﹏ | 文字左侧 | 波浪线（U+FE34） | 需处理连贯性 |
| 专名号 ＿ | 文字左侧 | 直线 | 人名、地名、朝代名 |
| 引号 「」『』 | 上下两端 | 居中 | 竖排通常使用角引号 |
| 破折号 | 居中，上下方向 | 占2字位，不得断开 | |
| 省略号 | 居中，上下方向 | 占2字位，不得断开 | |

### 3.2 标点占格的两种模式

古籍排版中标点占格存在两种传统：

#### 模式一：标点占格（现代点校本通行方式）
- 标点符号**占一个字的位置**
- 与正文字号相同
- 中华书局点校本二十四史等采用此方式

#### 模式二：标点不占格（传统古籍风格）
- 标点符号**不额外占字符位置**
- 标注在字的右下角（句读○），不影响排版密度
- 传统古籍刻本和仿古排版采用此方式
- 视觉效果更为饱满

> **中华书局实践**: 点校本二十四史等正式出版物采用**标点占格**方式，
> 但在追求传统版式的影印本和仿刻本中可能不占格。

## 四、排版软件中的竖排标点处理

### 4.1 方正书版

- 方正书版使用 BD 排版语言，共包括 136 个注解
- **SP 注解**: 专门用于竖排排版的命令
- **GZ 注解**: 割注（夹注）排版命令
- **CT 注解**: 词条排版命令（字典和古籍）
- 标点居右的处理通过字体和排版注解的组合实现

### 4.2 InDesign

- 使用"标点挤压设置"功能处理标点排版
- 竖排标点的位置通过字体的 `vert` / `vrt2` OpenType 特性实现
- 专名号可通过下划线功能模拟
- 书名号（波浪线）需要特殊处理以保证连贯性

### 4.3 字体选择对标点的影响

| 字体类型 | 标点位置 | 适用场景 |
|----------|----------|----------|
| 大陆字体（宋体、楷体等） | 偏左下 | 需要在竖排时通过 vert 特性转换 |
| 台湾字体（华康等） | 居正中 | 天然适合竖排 |
| 思源宋体 | 提供竖排变体 | 通过 OpenType vert 特性支持竖排 |
| 方正书版专用字体 | 内置竖排支持 | 专业排版 |

## 五、古籍排版中的特殊标点处理

### 5.1 夹注中的标点

- 夹注（双行小字注）中的标点通常使用较小字号
- 夹注中的标点仍遵循偏右（竖排）的位置规则
- 夹注中一般不使用书名号和专名号

### 5.2 校勘记中的标点

- 校记用方括号〔〕
- 校勘记格式示例："甲"，某本作"乙"。/ "甲"，原作"乙"，据某本改。
- 古籍排印本不可作为校本使用

### 5.3 句读号与现代标点的区别

| 传统句读 | 现代标点 | 说明 |
|----------|----------|------|
| ○（句） | 。（句号） | 传统句号为实心圆或空心圆 |
| 、（读） | ，（逗号） | 传统的"读"用于句内停顿 |
| 无 | ？！；：等 | 传统古籍不使用这些标点 |

## 六、点校本二十四史修订工程体例

### 6.1 基本信息

- 由中华书局主持的大型古籍整理项目
- 1963年后制定了适用于《晋书》以下二十史的标点和分段体例
- 修订工程在校勘理念、版本、文献等方面有很大进步

### 6.2 标点体例要点

- 采用现代标点符号替代传统句读
- 标点符号使用遵循 GB/T 15834 国家标准
- 专名号标在人名、地名等专有名词的左侧（竖排）
- 书名号使用波浪线形式标在文字左侧（竖排）
