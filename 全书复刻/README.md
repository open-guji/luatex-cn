# 全书复刻

本目录展示使用 luatex-cn 完整复刻的古籍原本项目。每一个项目都力求以像素级精度还原原书的排版风格、装饰元素和阅读体验。

## 《欽定四庫全書簡明目錄》— 文渊阁本

![四库全书](../示例/首页展示/mulu-color.png)

**原本介绍**
这是清朝编纂的最重要的古籍总目录，记录了四库全书所收的 3万+ 古籍的基本信息。版本采用文渊阁本，现藏台湾国立故宫博物院。

**版式特色**
- 🎨 **黄色封面** — 尊贵的宫廷配色
- 📐 **八行二十一字** — 经典的古籍开本
- 🏤 **"文渊阁宝"印章** — 御书房的身份标记
- 📄 **淡黄纸张** — 还原古籍的质感

**当前进度**
- ✅ **第一册完成** — 全页排版、校对、印章定位
- 📚 **9 册待续** — 共 10 册 20 卷的宏大工程

## 《紅樓夢》— 甲戌本（脂砚斋评本）

![红楼梦](../示例/首页展示/honglou-p1.png)

**原本介绍**
中国古代最伟大的小说，甲戌本是最珍贵的手抄本，保留了脂砚斋的眉批和侧批评注，是研究红楼梦文献的必读版本。

**版式特色**
- ✍️ **手抄本风格** — 原汁原味的古籍气韵
- 📝 **眉批与侧批** — 脂砚斋的精妙评注
- 💬 **脚注系统** — 详细的版本校记

**当前进度**
- ✅ **凡例 + 第一回** — 经典开篇已排版
- 📖 **后续章回** — 陆续加入中...
