# 示例目录

本目录包含 `luatex-cn` 宏包的多个使用示例，展示了从传统古籍排版到现代竖排的不同功能。

## 示例列表

### 1. [史记五帝本纪](./史记五帝本纪/)
- **特点**：演示了高度复杂的古籍排版。
- **功能**：
    - 绝对定位的红色印章（覆盖文字）。
    - 版心自定义文字与单鱼尾。
    - 复杂的加注（夹注）排版。
    - 传统的乌丝栏与排版缩进。

### 2. [史记目录](./史记目录/)
- **特点**：仿照清代《四库全书》北四阁本风格。
- **功能**：
    - 标准的古籍目录布局。
    - “八行二十一字”的行款限制。
    - 典型的白口、四周双边、单鱼尾版式。

### 3. [红楼梦甲戌本](./红楼梦甲戌本/)
- **特点**：模拟手抄本/批改本风格。
- **功能**：
    - **侧批与眉批**：利用 `sidenote` 系统实现的边缘注释。
    - **双列小字**：正文中的双行小字注。
    - 无鱼尾版心，底部页号显示。

### 4. [现代竖排书](./现代竖排书/)
- **特点**：简洁的现代竖排风格。
- **功能**：
    - 展示了不带古籍元素的纯粹竖排文本支持。
    - 适用于现代文学或报告的垂直排版需求。
    - 目前现代竖排书不是项目重点，后续会添加更多功能。

---

*注意：所有古籍示例均基于历史文献扫描件进行数字重构，旨在展示 `luatex-cn` 对各种复杂垂直排版场景的适配能力。*
